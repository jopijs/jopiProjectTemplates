# AGENTS

L'objectif de ce document est d'indiquer des instructions pour les agents.

----

## Documentation

* Quand tu génères du code serveur
  * Réfère-toi au fichier [ARCHITECTURE.md](_llm/ARCHITECTURE.md) pour comprendre l'architecture de l'application.
  * Réfère-toi au fichier [HOW-TO-SERVER.md](_llm/HOW-TO-SERVER.md) pour apprendre comment faire certaines tâches côté serveur.

* Quand tu génères du code pour le navigateur
  * Réfère-toi au fichier [HOW-TO-REACT.md](_llm/HOW-TO-REACT.md) pour apprendre comment faire certaines tâches en React.js.
  * Réfère-toi au fichier [HOW-TO-BROWSER.md](_llm/HOW-TO-BROWSER.md) pour les autres choses liées au navigateur.

* N'explore pas automatiquement les autres fichiers du dossier [_llm](_llm/). 

## Instructions

* Quand tu écris du code, celui-ci doit toujours utiliser des noms et des commentaires en anglais.

* Quand on te pose une question, alors si possible ajoute des suggestions venant des fichiers HOW TO.

* Après chaque question, indique quel fichier de documentation tu as lu afin de trouver tes réponses.
  Cette précision me permettra de savoir comment organiser cette documentation afin d'éviter que tu aies à lire trop de choses.
